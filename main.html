Virtual Memory Manager (VMM) Project link

This project implements a simple virtual memory manager (VMM) in Python. The VMM simulates the memory management techniques used by modern operating systems to manage processes and their memory allocations. Page Replacement Algorithms: Implements a simple page replacement algorithm to handle page faults. Multi-threaded Simulation: Uses threading to simulate concurrent execution of processes and memory management tasks. Address Translation: Translates virtual addresses to physical addresses using a page table. Memory Management: Manages memory allocation and deallocation for processes.
